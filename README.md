## 📖 Overview

**ProJob Portal** is a comprehensive platform designed to streamline the hiring process. Built with Python, Flask, and MySQL, it provides a seamless experience for both job seekers looking for their dream roles and employers searching for the perfect candidates. The frontend features a professional, responsive, and dynamic UI designed to wow users.

## ✨ Key Features

### 👨‍💻 For Job Seekers
* **Advanced Job Search:** Filter and find jobs based on technology, field, stipend, or salary expectations.
* **Easy Applications:** Apply to relevant positions quickly by submitting your resume and essential information.
* **Profile Management:** Keep your professional profile updated to attract top employers.

### 🏢 For Employers
* **Job Postings:** Create, edit, and manage detailed job listings, including requirements and contact details.
* **Application Management:** Review submitted applications for your job postings efficiently.
* **Direct Communication:** Communicate seamlessly with candidates (featuring integrated Flask-Mail support).

## 🛠️ Technologies Used

* **Backend:** [Python](https://www.python.org/) & [Flask](https://flask.palletsprojects.com/) - Providing a lightweight, robust, and scalable server infrastructure.
* **Frontend:** HTML5, modern CSS3 (with responsive design), and Vanilla JavaScript (for smooth animations and interactions).
* **Database:** [MySQL](https://www.mysql.com/) - A powerful relational database for secure storage of jobs, applications, and user profiles.

## 🚀 Getting Started

### Prerequisites
Make sure you have the following installed:
* Python 3.x
* MySQL Server
* pip (Python package installer)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Prathambathla10/jobportal.git
   cd jobportal
   ```

2. **Set up a virtual environment (recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Database Configuration:**
   * Create a MySQL database for the project.
   * Update the database connection settings in your `app.py` or environment variables to match your local MySQL credentials.

5. **Run the Application:**
   ```bash
   flask run
   ```
   *The application will be available at `http://127.0.0.1:5000/`*
